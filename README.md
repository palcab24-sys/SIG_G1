# Le Juste SIG - Gestion de Inventario

Sistema de informacion para administrar el inventario fisico y los productos virtuales de Le Juste, usando MySQL y Python con PyQt5.

## Archivos principales

- `lejuste_db.txt`: script para crear la base de datos en MySQL.
- `app3.py`: aplicacion Python/PyQt5 para gestionar inventario y pedidos.
- Archivos `.ui`: interfaces graficas del sistema.

## Requisitos
- Python 3.x
- PyQt5 (`pip install pyqt5`)
- pymysql (`pip install pymysql`)
- MySQL
## Instalacion y uso

1. Pegar `lejuste_db.txt` en su servidor MySQL.
2. Ejecuta el archivo Python, ingresa las credenciales de base de datos en la configuracion y utiliza la aplicacion para ver y editar el inventario.
